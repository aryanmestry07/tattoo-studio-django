# Tattoo Studio Booking System

A full-stack Django web application for managing a tattoo studio.

## Features
- User authentication (login/signup)
- Tattoo gallery
- Artist management
- Slot-based booking system
- Admin dashboard
- Review system

## Tech Stack
- Django
- Bootstrap
- SQLite

## Setup

git clone <repo-url>
cd project
pip install -r requirements.txt
python manage.py runserver
